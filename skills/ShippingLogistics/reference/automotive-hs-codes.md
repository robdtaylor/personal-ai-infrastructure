## Classification Tips

### General Rules

1. **Specificity**: More specific description prevails over general
2. **Essential Character**: Composite goods classified by essential character
3. **Last Heading**: When rules 1-3 don't apply, use last applicable heading
4. **Parts**: Parts suitable solely/principally for a machine = classified with that machine

### Common Misclassifications

| Often Misclassified | Correct Classification |
|--------------------|----------------------|
| Rubber hoses with fittings | 4009 (not 8708) |
| Wiring harnesses | 8544.30 (not 8708) |
| Electronic control units | 9032.89 (not 8708) |
| Bearings | 8482 (not 8708) |
| Fasteners | 7318 (not 8708) |

### When to Use 8708

Use 8708 ONLY when:
- Part is identifiable as solely/principally for motor vehicles
- Part is not covered more specifically elsewhere
- Part is not a general-purpose article
