# X-Matrix - Excel Paste Format

## Instructions
1. Copy the tab-separated tables below
2. Paste directly into Excel (Ctrl+V / Cmd+V)
3. Data will auto-populate into columns
4. Format as needed

---

## BREAKTHROUGH OBJECTIVES (3-5 Year) - Paste into Row 1

