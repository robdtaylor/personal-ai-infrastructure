# Campaign Workflow

Plan and execute marketing campaigns with structured timelines.

## Campaign Types

| Type | Duration | Primary Goal |
|------|----------|--------------|
| Product Launch | 4-8 weeks | Awareness + Sales |
| Promotion | 1-2 weeks | Sales spike |
| Content Series | 4-12 weeks | Engagement + Leads |
| Event | 2-4 weeks | Registrations |
| Rebrand | 8-12 weeks | Awareness |
| Seasonal | 2-4 weeks | Sales |

## Campaign Framework

### Phase 1: Strategy (Week -4 to -2)

```markdown
## Campaign Brief

**Campaign Name:** [Name]
**Type:** [Launch/Promo/Content/Event]
**Duration:** [Start] to [End]

### Objectives
- Primary: [Specific, measurable goal]
- Secondary: [Supporting goal]

### Target Audience
- Segment: [Who specifically]
- Pain Point: [What problem]
- Desire: [What outcome they want]
- Awareness Level: [Unaware/Problem/Solution/Product/Most Aware]

### Key Messages
1. [Primary message/value prop]
2. [Supporting message]
3. [Differentiator]

### Offer
- What: [Product/service/lead magnet]
- Price/Value: [Cost or value]
- Urgency: [Deadline/scarcity]
- Guarantee: [Risk reversal]

### Success Metrics
| Metric | Target | Tracking Method |
|--------|--------|-----------------|
| [Metric] | [Goal] | [Tool/method] |

### Budget
| Category | Amount |
|----------|--------|
| Paid ads | $ |
| Content | $ |
| Tools | $ |
| Total | $ |
```

### Phase 2: Content Creation (Week -2 to -1)

```markdown
## Content Checklist

### Core Assets
- [ ] Landing page copy
- [ ] Sales page (if applicable)
- [ ] Email sequence ([X] emails)
- [ ] Social media posts ([X] posts)
- [ ] Ad creative ([X] variations)

### Supporting Assets
- [ ] Blog post(s)
- [ ] Lead magnet
- [ ] Case study/testimonial
- [ ] FAQ document
- [ ] Graphics/visuals

### Technical Setup
- [ ] Landing page built and tested
- [ ] Email automation configured
- [ ] Tracking pixels installed
- [ ] UTM parameters defined
- [ ] A/B tests configured
```

### Phase 3: Pre-Launch (Week -1)

```markdown
## Pre-Launch Checklist

### Internal
- [ ] Team briefed on campaign
- [ ] Support team has FAQ
- [ ] Sales team has materials
- [ ] Stakeholders approved

### Technical
- [ ] All links tested
- [ ] Forms working
- [ ] Payment processing tested
- [ ] Mobile responsiveness checked
- [ ] Load testing complete

### Teaser Content
- [ ] "Coming soon" posts scheduled
- [ ] Email list teased
- [ ] Early access offered to VIPs
```

### Phase 4: Launch (Week 0)

```markdown
## Launch Day Checklist

### Morning
- [ ] Final link checks
- [ ] Team standup
- [ ] Monitoring dashboards ready

### Launch
- [ ] Landing page live
- [ ] Email #1 sent
- [ ] Social posts published
- [ ] Paid ads activated
- [ ] PR/outreach sent

### Monitoring
- [ ] Check every 2 hours
- [ ] Respond to comments/questions
- [ ] Address technical issues immediately
- [ ] Document learnings
```

### Phase 5: Sustain (Week 1-X)

```markdown
## Ongoing Activities

### Daily
- Social media engagement
- Email performance check
- Customer support monitoring
- Quick wins documentation

### Weekly
- Performance review
- Content refresh
- A/B test analysis
- Budget reallocation

### Communications Calendar
| Day | Channel | Content |
|-----|---------|---------|
| Mon | Email | [Topic] |
| Tue | Social | [Topic] |
| Wed | Blog | [Topic] |
| ... | ... | ... |
```

### Phase 6: Close (Final Week)

```markdown
## Closing Sequence

### Urgency Escalation
- Day -7: "One week left"
- Day -3: "Final days"
- Day -1: "Last chance"
- Day 0: "Closes tonight"

### Post-Campaign
- [ ] Final results compiled
- [ ] Retrospective meeting
- [ ] Learnings documented
- [ ] Assets archived
- [ ] Follow-up sequence activated
```
