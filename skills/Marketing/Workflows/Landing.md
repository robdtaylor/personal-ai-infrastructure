# Landing Page Workflow

Create high-converting landing page copy.

## Page Types

| Type | Purpose | Key Sections |
|------|---------|--------------|
| Lead Magnet | Capture emails | Hero, Benefits, Form |
| Sales | Sell product/service | Full persuasion arc |
| Webinar | Event registration | Hero, Agenda, Speaker, Form |
| Waitlist | Build anticipation | Hero, Tease, Form |
| Comparison | Beat competitors | Hero, Table, Proof, CTA |

## Landing Page Anatomy

### Above the Fold (Critical)

```markdown
## Hero Section

### Headline
[Benefit-driven, 10 words max]
Formula: "Get [Benefit] Without [Pain Point]"

### Subheadline
[Expand on headline, handle objection]
Formula: "[How it works] so you can [benefit]"

### Hero Image/Video
[Show the transformation or product in action]

### Primary CTA
[Action verb + benefit]
"Start [Benefit] Free" / "Get My [Resource]"

### Trust Indicators
[Logos, numbers, mini-testimonials]
"Trusted by 10,000+ [audience]"
```

### Problem Section

```markdown
## The Problem

### Headline
"If you're struggling with [problem], you're not alone"

### Pain Points (3-5)
- [Specific frustration #1]
- [Specific frustration #2]
- [Specific frustration #3]

### Agitation
"Without solving this, [consequence]"
```

### Solution Section

```markdown
## The Solution

### Headline
"Introducing [Product]: [One-line value prop]"

### How It Works (3 steps)
1. [Simple step] → [Outcome]
2. [Simple step] → [Outcome]
3. [Simple step] → [Outcome]

### Key Benefits (not features!)
- [Benefit] (not "Feature X")
- [Benefit] (not "Feature Y")
- [Benefit] (not "Feature Z")
```

### Social Proof Section

```markdown
## Proof

### Testimonials
[Name, Title, Company]
"[Specific result with numbers]"

### Case Study Highlight
"[Company] increased [metric] by [%] in [time]"

### Trust Badges
- Logos of known clients
- Media mentions
- Certifications
- Security badges

### Numbers
- [X]+ customers
- [Y]% satisfaction
- [Z] years in business
```

### Objection Handling

```markdown
## FAQ / Objections

### Common Objections
Q: "Is this right for [my situation]?"
A: [Reassurance + proof]

Q: "What if it doesn't work?"
A: [Guarantee/risk reversal]

Q: "Why is it priced this way?"
A: [Value justification]

Q: "How is this different from [competitor]?"
A: [Differentiator]
```

### Final CTA Section

```markdown
## Final Push

### Recap
"Here's everything you get..."

### Price/Value Stack
- [Component 1] (Value: $X)
- [Component 2] (Value: $Y)
- Total Value: $[Total]
- Your Price: $[Price]

### Urgency (if authentic)
"[Deadline/Scarcity element]"

### Guarantee
"[Time]-day money-back guarantee"

### Final CTA
[Same as hero, slightly more urgent]
"[Action] Now - [Urgency]"
```

## Framework Applications

### AIDA Landing Page
```
Hero = Attention
Problem + Solution = Interest
Benefits + Proof = Desire
CTA sections = Action
```

### PAS Landing Page
```
Hero = Problem teaser
Problem section = Problem + Agitate
Solution + Proof + CTA = Solution
```

### Long-Form Sales Page (PASTOR)
```
Hero = Problem hint
Problem section = Problem + Amplify
Story section = Story
Transformation = Testimonials + Results
Offer section = Offer details
Multiple CTAs = Response
```

## Conversion Elements

### Trust Builders
```
- Specific numbers ("10,847 customers" not "thousands")
- Real names and photos in testimonials
- Verifiable credentials
- Security badges (SSL, payment processors)
- Money-back guarantee
- Live chat availability
```

### Friction Reducers
```
- Short forms (name + email only for leads)
- "No credit card required"
- Clear pricing (no surprises)
- Fast load time
- Mobile optimization
```

### Urgency (Use Ethically)
```
- Real deadlines
- Genuine scarcity
- Bonus expiration
- Price increase dates
- Avoid fake countdown timers
```
