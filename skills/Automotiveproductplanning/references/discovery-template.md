# Company Discovery Template

Structured questions to capture company profile and strategic context before product planning.

## Instructions

Work through each section, capturing responses. Skip sections not applicable to the engagement. Use responses to:
- Recommend appropriate planning horizon
- Identify constraints and opportunities
- Tailor analysis frameworks
- Select relevant output formats

---
