# Kai Todo List Manager

Manage Kai's task queue for work to do when Claude usage is available.

## Instructions

Read the skill at `~/.claude/Skills/KaiTodo/SKILL.md` for detailed instructions, then execute the requested command.

**Arguments**: `$ARGUMENTS`

## Quick Reference

- `add <topic>` - Add task to queue
- `list` or empty - Show queued tasks
- `next` - Start highest priority task
- `done <id>` - Mark task complete
- `remove <id>` - Delete task
